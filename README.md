# Plop

## Description
Have you ever needed to go to the restroom, but didn't know where the closest one was? Did you also want to know if it was clean enough?
Well... Plop is designed for you!

Discover, rate, and explore restrooms near you so you can do your stuff with peak performance and privacy.

## Features
- Ranking restrooms
	- cleanliness
	- maintained?
	- stank
- Features of restroom	
- Images
- Directions/Routing (map)
- Nearby (map)

## Tech Stack
- React
- Firebase
